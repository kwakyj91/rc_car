 🏎️ STM32 협소 트랙 자율주행 Car 프로젝트


  본 프로젝트는 STM32F411RETX를 기반으로 3개의 초음파 센서를 활용하여 좁고
  복잡한 트랙을 자율 주행하는 스마트 카 시스템입니다. 차체 크기 대비 트랙 폭이
  매우 좁은 극한 환경에서도 벽과의 충돌을 방지하고 안정적인 주행을 구현하는 데
  집중했습니다.

  📌 주요 특징 (Key Features)


   * 듀얼 모드 지원: 블루투스 통신을 통한 수동 제어 모드와 초음파 센서 기반 자율
     주행 모드 지원.
   * 3방향 초음파 센서 정렬: 정면(0°) 및 좌우 대각선(45°) 배치로 사각지대
     최소화.
   * 협소 트랙 최적화 알고리즘: 차체 길이(27cm)가 트랙 폭(39cm)의 70%에 달하는
     환경에서도 회전이 가능한 정밀 제어 로직 적용.
   * 비선형 벽 충돌 방지 (Wall Repulsion): 벽과의 거리가 임계값 이하로 좁아질
     경우 강력한 반대 방향 회전력을 부여하는 회피 기동.

  🛠️ 하드웨어 구성 (Hardware)


   * MCU: STM32F411RE (Cortex-M4)
   * Motor Driver: L298N (DC Motor x2)
   * Sensors: HC-SR04 초음파 센서 x3 (좌/중/우)
   * Communication: HC-06 Bluetooth 모듈
   * Chassis: 차체 길이 27cm, 폭 17cm

  🧠 핵심 알고리즘 (Algorithm Highlights)


  1. 센서 간섭 제거 (Sequential Triggering)
  초음파 센서 3개를 동시에 트리거할 때 발생하는 신호 간섭(Crosstalk)을 방지하기
  위해, 각 센서를 20ms 간격으로 순차 트리거하는 스케줄링 로직을 구현했습니다.
  (전체 측정 주기: 60ms)


  2. 정밀 중앙 유지 (Lane Keeping)
  좌우 센서값의 차이(diff = L - R)를 계산하여 모터 속도를 실시간으로 보정합니다.
   * 선형 보정: 일반 주행 시 부드러운 조향 유지.
   * 비선형 보정: 벽과의 거리가 11cm 이하로 좁아질 경우 보정치를 최대화하여
     즉각적으로 벽에서 멀어지도록 설계.


  3. 직각 코너 및 모서리 탈출 (Escape Logic)
   * Stop-and-Pivot: 정면 장애물 감지 시 관성을 제어하기 위해 완전 정지 후
     제자리 회전 수행.
   * 후진 시퀀스: 회전 공간이 부족한 협소 구간에서 짧은 후진(0.4s) 후 공간을
     확보하여 회전하는 시퀀스 제어.
   * 역주행 방지 (Turn Cool-down): 코너 회전 직후 0.8초간 조향 판단을 유예하여
     차체가 트랙 방향으로 완전히 정렬되도록 유도.


  4. 센서 데드존(Deadzone) 처리
  초음파 센서가 벽에 너무 밀착(2.5cm 이하)되어 값이 튀는 현상을 방지하기 위해,
  데드존 발생 시 해당 방향 거리를 0으로 강제 처리하여 장애물이 없는 것으로
  착각하지 않도록 안전 장치를 마련했습니다.

  📺 주행 로직 흐름도


   1. 센서 데이터 수집: 좌(45°), 중(0°), 우(45°) 순차 측정.
   2. 모드 판단: 자동(drive_mode 1) / 수동(drive_mode 0).
   3. 상태 체크: 전방 장애물 유무 및 측면 근접도 확인.
   4. 모터 제어: 상황에 따른 전진/후진/피벗 회전 및 PWM 속도 보정.
